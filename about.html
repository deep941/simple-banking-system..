Simple Banking System

About Our Banking System

Welcome to the Simple Banking System! This is a basic web application that allows users to create and manage their bank accounts, make deposits, withdraw money, and check their account balances.

This project was developed to demonstrate how a basic banking system can be implemented using Python and Flask for the backend, along with HTML and CSS for the frontend. It's a learning project aimed at students who want to explore how web applications work and how to integrate a simple user interface with server-side logic.

Key Features

  • Create new bank accounts
  • Make deposits and withdrawals
  • Check account balances

Technologies Used

  • Backend: Python, Flask
  • Frontend: HTML, CSS
